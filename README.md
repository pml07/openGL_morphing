# OpenGL_morphing
###### tags: `openGL`

### Environment & Libraries
---
* OS：Windows 10
* python 3.7.2
* pyQt5
* opencv


### 目標
---
* 實作 Feature-Based Image Metamorphosis
* Input:
    * 2-3 color images
    * feature lines segments
    * alpha value
* Output:
    * single warped image (before color blending)
    * final warping result (after color blending)
